# 基于SpringBoot+Vue.js的RABC权限系统
---

## 采用技术：
SpringBoot+SpringDataJPA+Springsecurity+JWT+vue.js+elementUI

## 你将学到
---

- Spring Boot 多模块项目的创建
- Vue.js 整合 ElementUI 前端项目的搭建
- Spring Seurity+JWT 的认证和鉴权
- 真实企业开发前后端分离数据的封装
- 实操一个多模块的前后端分离的 RABC 权限系统
